# Detris
Наверняка некоторым иногда надоедают игры со сложной графикой, логикой и игровым интерфейсом, но и известные всем мини-игры (flappy bird, pac-man и др.) уже наскучили. Что ж, Detris решает это! 
# -
Detris - мини-игра, сочетающая в себе Tetris и элементы "выживания", так сказать. Простота игры помогает немного отвлечься от сложных эпичных игр. 
# Механика игры
Вы играете за маленького шахтёра, задача которого - уворачиваться от летящих сверху блоков, засчёт передвижения при помощи стрелок на клавиатуре. Если блок упал, на него можно залезть. Зачем? Например чтобы собрать монетки, летящие сверху, чтобы получить +5 к счёту. 
Счёт по умолчанию прибавляется за кажую секунду выживания в игре. Каждые 100 очков (ну, почти каждые) выводится поддерживающая фраза.





отвечу честно, Detris'а два. Один - разработанный для защиты (по исправлению тестировщика) и один полностью разработчика, решившего отклониться от команды и разработать свой, хотя они похожи.
